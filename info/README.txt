# GestionHotel

Descripción
GestionHotel es un programa hecho para almacenar y gestionar información; Además, nos permite agilizar procesos. Todo esto se habla en un contexto de un hotel. Por último, esto nos permite tener un mejor orden en cuanto a la información que vayamos a manejar y que tan eficiente queremos esa información.

Los métodos implementados son:
-Calcular total a pagar de un cliente por una noche en el hotel.
-Obtener una lista de habitaciones según su tipo.
-Obtener la información de un cliente por su dni

Este programa se hizo, para reforzar los conceptos y principios SOLID que debemos manejar y usar cada vez que vayamos a realizar código.
